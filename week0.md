# Binary, Bytes, and Basic Coding Concepts

## Bits and Bytes

A **bit** (binary digit) can contain only `0` or `1`.

A **byte** consists of 8 bits.

Each digit position represents a power of two:

```
1  2  4  8  16  32  64  128
```

Each position answers the question: *"How many of this number do we have in our target number?"*

A byte can look like `00000000` and can represent 256 different values, from `0` to `255`.

## Text Encoding

**ASCII** contains patterns for the alphabet and other characters/symbols, encoded as bytes. We can easily write text messages using these patterns.

**Unicode** is used for emojis (and other extended characters) and can use multiple bytes per character.

## Colors and Images

**RGB** is used for colors and uses Red, Green, and Blue values — each of them stored in one byte.

Each part of an image is called a **pixel**, and it contains RGB information about the color of that exact pixel.

## What Is Code?

Code is literally a way to tell the computer an algorithm.
